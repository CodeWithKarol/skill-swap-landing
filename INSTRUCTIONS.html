‚úèÔ∏è Modifying Content

Page Title & Meta Description

index.html (lines 5-7)
<title>SkillSwap Hub - Trade Skills, Expand Horizons</title> <meta name="description" content="Your description here">

Edit the <title> tag to change the browser tab title and search engine results.

Header/Navigation Text

index.html (lines 19-140)

Find the <header class="main-header"> section to modify:

  • Logo text: Search for <span class="logo-text">
  • Navigation links: Look for <nav class="main-nav">
  • Button text: Search for class="btn-header"

Section Headers & Descriptions

Each section has a consistent structure:

<div class="section-header"> <div class="section-badge">...</div> <h2 class="section-title">Your Title Here</h2> <p class="section-description">Your description</p> </div>

Search for section-title or section-description to find all sections.

index.html (lines 240-350)

  • Main headline: "Exchange Skills, Build Connections, Grow Together"
  • Description: Main value proposition
  • CTA buttons: "Get Started Free" and "Watch Demo"
  • Feature items: Three highlighted features

Testimonials

index.html (search for "testimonial-card-modern")

To modify testimonials:

  1. Find the <div class="testimonial-card-modern"> section
  2. Change the <p class="testimonial-text"> content
  3. Update author name in <h4 class="author-name">
  4. Update author role in <p class="author-role">
  5. Change avatar image URL (see Images section below)
  6. Modify skill tags in <span class="testimonial-skill-tag">

Scroll Behavior

The mobile menu uses a sophisticated scroll prevention system. Key variables:

  • menuState - Tracks menu open/close state
  • menuState.isAnimating - Prevents scroll during animations
  • menuState.scrollPosition - Stores scroll position when menu opens

The mobile menu appears below 767px screen width. To change this:

// In styles.css, search for: @media (max-width: 767px) // In script.js, search for: if (window.innerWidth > 767)

Option 2: GitHub Pages

  1. Push your files to a GitHub repository
  2. Go to Settings ‚Üí Pages
  3. Select main branch as source

Option 3: Traditional Hosting

  1. Upload all files to your web hosting via FTP
  2. Make sure index.html is in the root directory
  3. Site will be live at your domain

‚ùì Troubleshooting

Mobile Menu Not Working

  • Check browser console (F12) for errors
  • Verify .mobile-menu-toggle element exists in HTML
  • Ensure script.js is properly linked

Styles Not Applying

  • Verify styles.css is linked in the <head>
  • Clear browser cache (Ctrl+Shift+Delete)
  • Check CSS class names match HTML

Images Not Showing

  • Check image URLs are correct
  • Verify images exist at specified paths
  • Check image file permissions

Smooth Scroll Not Working

  • Some older browsers don't support behavior: smooth
  • Check JavaScript console for errors
  • Verify anchor links have correct href values
